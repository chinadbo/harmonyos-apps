# HarmonyPlayer

HarmonyPlayer 是一个基于 HarmonyOS API 16 的本地音乐播放器示例工程，覆盖本地扫描、AVPlayer 播放、AVSession 媒体会话、InsightIntent 语音控制、跨设备流转与桌面 Widget 等能力。

## 技术架构图

```text
+----------------------- HarmonyPlayer -----------------------+

                             |

            |              |

    +------------+   +----------------------+
            |
    +-------+-------------------------------+

    +---------------------------------------+
```

## 核心功能列表

- 本地音乐扫描与搜索
- AVPlayer 播放控制、进度拖拽、上下曲切换
- 播放队列与播放模式切换
- 滚动歌词与旋转专辑封面
- 收藏与播放历史持久化接口
- 后台播放与 AVSession 系统媒体控制

## 鸿蒙特色能力

- InsightIntent：支持播放、暂停、上下曲等语音控制入口
- AVSession：系统媒体控制中心元数据与播放状态同步
- 跨设备流转：序列化当前歌曲与进度以支持 Continuation
- 桌面 Widget：2x4 卡片显示当前播放并提供播放控制

## 构建方式

1. 使用 DevEco Studio 4.1+ 打开本工程目录。
2. 确认 SDK 版本支持 HarmonyOS API 16。
3. 选择 `default` product 后构建 `entry` 与 `widget` 模块。
4. 如需真机调试，请在签名配置中补充正式签名材料。
