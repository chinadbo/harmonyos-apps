# 今日选题：HarmonyHabit — 习惯追踪与打卡

**日期：** 2026-05-06  
**品类标签：** 习惯追踪/打卡  
**复杂度评估：** 中（目标 60min 完成主体）

## App 定位

面向注重自我管理的 HarmonyOS 用户，提供**可视化习惯追踪与每日打卡**功能。
核心差异化：桌面 Widget 一键打卡 + reminderAgentManager 定时提醒 + 热力图统计视图。

## 核心功能

| 功能模块 | 描述 |
|----------|------|
| 习惯管理 | 创建/编辑/删除习惯，支持图标、颜色、目标频率（每日/指定天） |
| 每日打卡 | 首页列表，当日未打卡高亮，一键完成打卡 |
| 打卡日历 | 单个习惯的月度热力图，展示坚持情况 |
| 统计概览 | 当日完成率、连续打卡天数（streak）、本周完成率 |
| 定时提醒 | 每个习惯可设置提醒时间，基于 reminderAgentManager CALENDAR 类型 |
| 桌面 Widget | 2×2 卡片展示今日待打卡数量 + 完成数，支持点击跳转 |

## 技术亮点

1. **reminderAgentManager** — 精确时间提醒，支持 CALENDAR 重复提醒，最多 30 个（含 evictOldest 策略）
2. **RelationalStore (RDB)** — 习惯表 + 打卡记录表，支持按日期范围查询统计
3. **Widget (FormExtensionAbility)** — 2×2 桌面卡片，postCardAction 跳转首页
4. **热力图组件** — 纯 ArkUI Canvas 绘制，无第三方库依赖
5. **V1 装饰器体系** — @State/@Prop/@Link 数据流，AppStorage 跨页状态

## 页面列表

1. **首页 (Index)** — 今日习惯打卡列表
2. **统计页 (Statistics)** — 概览 + 各习惯 streak
3. **习惯详情 (HabitDetail)** — 月度热力图 + 历史打卡记录
4. **习惯编辑 (HabitEdit)** — 创建/编辑表单
5. **设置页 (Settings)** — 通知权限、主题等
