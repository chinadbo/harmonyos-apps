## 隐私声明

**鸿蒙翻译（HarmonyTranslate）** 是一款多语言翻译工具。本应用严格遵守隐私保护原则，所有用户数据均存储在本地设备上，不会上传至任何服务器。

### 数据收集
- 本应用仅收集用户主动输入的翻译文本
- 翻译历史记录存储在本地数据库中
- 所有数据均存储在设备本地，不会同步到云端

### 权限使用
- **INTERNET**：用于调用在线翻译API（MyMemory Translation API）
- 不访问用户联系人、照片、位置等敏感信息

### 第三方服务
- 使用 MyMemory Translation API 提供在线翻译服务
- 该服务为免费翻译API，不涉及用户数据收集

### 数据安全
- 所有数据存储在本地SQLite数据库中
- 应用卸载后所有数据将被删除
